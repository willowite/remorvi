# Бренд-ассеты Remorvi: условия использования

Все файлы в этой папке (`gh-hero-banner.jpg`, `gh-detect-clean.jpg`, `gh-how-it-works.jpg`,
`remorvi-loop.mp4`, `remorvi-loop.gif`, `remorvi-poster.jpg`, `favicon.png` и производные) –
визуальная идентичность бренда **Remorvi**: кольцо-марка, палитра, оформление.

**© Remorvi. Все права защищены.** Это НЕ распространяется под лицензией кода репозитория.

Нельзя без письменного разрешения: использовать марку/ассеты в своих продуктах, лендингах,
рекламе; изменять цвета/форму марки; создавать производные, вводящие в заблуждение о связи с Remorvi.

Можно: ссылаться на проект со скриншотом README в обзорах/статьях с указанием источника.

Контакт по бренду: [@remorvibot](https://t.me/remorvibot).
